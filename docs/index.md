# Ingot 文档

Ingot 将不同数据源转换为标准生产事件，并在 Central 中形成可信事实。Central Web 的 **Chat** 用于只读查询和问题定位；下载安装的 **Ingot Agent** 桌面端只用于连接器代码生成、构建、测试和打包。
